Quick Start

Running in 4 steps.

01
Setup on Colab T4 GPU
Run the one-click setup notebook. Installs all dependencies and downloads model weights automatically. Takes 3–5 minutes once per session.
Runtime → Change runtime type → T4 GPU
▶ Run Cell 1 (Setup)
02
Load the V16 Engine
Initialise the CinematicEngineV16 instance with all 15 architectural improvements — PipelineManager, VRAMManager, SmartCache, and more.
▶ Run Cell 2 (Engine)
# engine = CinematicEngineV16()
03
Start the WebSocket Bridge
Launch the async bridge server. It broadcasts real GPU metrics every 2 seconds and handles task execution commands from the Dashboard.
▶ Run Cell 3 (Bridge)
# Listening on ws://localhost:7860
04
Open the Dashboard
Open the V17.2 HTML file in your browser. Click "WS: OFF" in the header to connect. The Dashboard lights up with live data immediately.
Open: cinematic_engine_v17_2_final.html
Click: WS: OFF → WS: LIVE ✓
